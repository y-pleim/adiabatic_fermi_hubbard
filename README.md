A Python package to simulate the Hubbard model ground state for simple lattices through adiabatic state preparation.

### Dependencies

This package has the following dependencies:
- numpy
- qiskit
- qiskit-nature
- qiskit-algorithms
- qiskit-aer
- matplotlib (optional)
