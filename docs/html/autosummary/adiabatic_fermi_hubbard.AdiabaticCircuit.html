adiabatic_fermi_hubbard.AdiabaticCircuit

class adiabatic_fermi_hubbard.AdiabaticCircuit(ham: HubbardHamiltonian, time_step: float = 0.1, step_count: int = 10000)[source]
__init__(ham: HubbardHamiltonian, time_step: float = 0.1, step_count: int = 10000)[source]

A class for building/executing Qiskit circuits for adiabatic state preparation of the Fermi-Hubbard Hamiltonian ground state through interpolating between \(H_{initial}\) and \(H_{final} = H_{Fermi-Hubbard}\) according to

\[H(k) = H_{initial} (1-k/M) + H_{final} (k/M).\]

where \(M\) is the number of interpolation steps, \(k = 0, 1, ... , M\), and \(H_{initial} = \sum_i^{2N} X_i\). \(N\) is the number of lattice sites.

Parameters:

  • ham (HubbardHamiltonian) – The HubbardHamiltonian object to find the ground state for.

  • time_step (float) – Duration of each interpolating step.

  • step_count (float) – The number of steps to take when interpolating between \(H_{initial}\) and \(H_{Fermi-Hubbard}\).
