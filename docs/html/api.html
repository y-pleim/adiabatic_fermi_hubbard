class adiabatic_fermi_hubbard.HubbardHamiltonian(lattice: Lattice, t: float = 2, U: float = 10, mu: float = -5)[source]
__init__(lattice: Lattice, t: float = 2, U: float = 10, mu: float = -5)[source]

A class for constructing the Fermi-Hubbard Hamiltonian for particular lattice, hopping strength \(t\), interaction strength \(U\), and chemical potential \(\mu\).

\[H = -t \sum_{<i,j>,\sigma}(a_{i\sigma}^\dagger a_{j\sigma} + h.c.) + U\sum_{i} n_{i\uparrow}n_{i\downarrow} + \mu \sum_{i,\sigma} n_{i\sigma}\]
Parameters:

  • lattice (Lattice) – The Lattice to find the Hamiltonian for.

  • t (float) – The strength of the hopping term.

  • U (float) – The strength of the onsite interaction term.

  • mu (float) – The chemical potential.

class adiabatic_fermi_hubbard.AdiabaticCircuit(ham: HubbardHamiltonian, time_step: float = 0.01, step_count: int = 1000)[source]
__init__(ham: HubbardHamiltonian, time_step: float = 0.01, step_count: int = 1000)[source]

A class for building/executing Qiskit circuits for adiabatic state preparation of the Hubbard Hamiltonian ground state through interpolating between \(H_{init}\) and \(H_{Fermi-Hubbard}\) according to

\[H(k) = H_{init} (1-k/M) + H_{final} (k/M).\]

where \(M\) is the number of interpolation steps, \(k = 0, 1, ... , M\), and \(H_{init} = \sum_i X_i\)

Parameters:

  • ham (HubbardHamiltonian) – The HubbardHamiltonian object to find the ground state for.

  • time_step (float) – Duration of each interpolating step.

  • step_count (float) – The number of steps to take when interpolating between \(H_{init}\) and \(H_{Fermi-Hubbard}\).

where \(H_{init} = \sum_i{X_i}\), \(H_{FH}\) is the Jordan-Wigner transformed Fermi-Hubbard Hamiltonian and \(M\) is the number of interpolation steps.

Parameters:

ising_setup(exchange)[source]

Support method that assigns an Ising Hamiltonian (for the same number of qubits) to the AdiabaticCircuit object:

\[H = J\sum_{<i,j>} Z_i Z_j\]

where \(<i,j>\) denotes nearest neighbors on the lattice. Implemented with periodic boundary conditions if the Lattice associated with the AdiabaticCircuit has periodic boundary conditions.

Parameters:
